# Пиксельная игра с вопросами

Многопользовательская игра, где игроки отвечают на вопросы и окрашивают пиксели на общем поле.

## Функциональность

- Многопользовательский режим в реальном времени
- Поле 10×10 пикселей
- Система вопросов и ответов
- Таблица лидеров
- Панель администратора для управления контентом

## Установка

1. Клонируйте репозиторий
2. Настройте Firebase:
   - Создайте проект в Firebase Console
   - Включите Realtime Database
   - Замените конфигурацию в `script.js`
3. Разместите на GitHub Pages

## Использование

- Обычный пользователь: введите ник и начинайте играть
- Администратор: используйте ник "админ123" для доступа к панели управления

## Технологии

- HTML5, CSS3, JavaScript
- Firebase Realtime Database
- GitHub Pages для хостинга
